Types of Colors

Color Red

Red

The color red is a named color in CSS. It can be represented using the keyword "red."

Color #104b97

#104b97

#104b97 is a hexadecimal color code representing a shade of blue. Hexadecimal colors are a common way to specify colors in web design.

Color RGB

RGB(60, 179, 113)

RGB (Red, Green, Blue) is another way to specify colors in CSS. Here, we use RGB(60, 179, 113) to create a shade of green.

Color RGBA

RGBA(255, 0, 0, 0.5)

RGBA (Red, Green, Blue, Alpha) is similar to RGB but includes an alpha channel for transparency. The "a" in RGBA stands for "alpha," which represents the transparency of the color. In this example, rgba(255, 0, 0, 0.5) creates a partially transparent red color with 50% opacity.
