CSS Text Shadow, Box Shadow, and Text Indent

Text Shadow

Text Shadow Example

<p style="text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);">Text Shadow Example</p>

The CSS text-shadow property is used to add a shadow to text. It takes parameters like the horizontal and vertical offset, blur radius, and color.

Box Shadow

<div style="width: 150px; height: 150px; background-color: #0077b6; box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);"></div>

The CSS box-shadow property adds shadow to elements like divs, buttons, and more. You can specify the horizontal and vertical offset, blur radius, spread, and color.

Text Indent

This is an example of text with an indent. Text is pushed in by 20px.

<p style="text-indent: 20px;">This is an example of text with an indent. Text is pushed in by 20px.</p>

The CSS text-indent property is used to specify the indentation of the first line in a text block. It can be used with paragraphs and other text elements.
