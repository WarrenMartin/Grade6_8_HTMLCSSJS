CSS Transform Property

RotateX Transformation

⬆(uparrow)

The CSS rotateX transformation is used to rotate an element around its horizontal axis. In this example, the element is rotated by 180 degrees along the X-axis, which flips it vertically.

RotateY Transformation

→(rightarrow)

The CSS rotateY transformation is used to rotate an element around its vertical axis. In this example, the element is rotated by 180 degrees along the Y-axis, which flips it horizontally.

            <div class="transform-example" style="transform: rotateX(0deg);">⬆(uparrow)</div>
            <div class="transform-example" style="transform: rotateY(0deg);">→(rightarrow)</div>
