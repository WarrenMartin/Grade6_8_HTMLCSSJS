Relative vs. Absolute Positioning in CSS

Relative Positioning

With position: relative, elements are adjusted relative to their original position on the page. They can be moved using properties like top, bottom, left, or right

Relative Position

Absolute Positioning

On the other hand, position: absolute treats elements as detached from the normal document flow. They can be precisely placed based on their nearest positioned ancestor or the page's edges, without impacting the layout of other elements.

Absolute Position

Key Differences

  • Relative: Elements are adjusted relative to their original position, and they affect the layout of other elements.
  • Absolute: Elements are detached from the normal flow and do not affect the layout of other elements.

Code Demo

Here's a code example demonstrating the difference between relative and absolute positioning:


.relative-demo {
    position: relative;
    left: 10px; /* Move 10 pixels to the right */
}

.absolute-demo {
    position: absolute;
    top: 20px;
    right: 20px;
}
