Understanding the Bootstrap Grid System

In Bootstrap, the grid system allows you to create responsive layouts for your web page. Let's explore how different column classes work for various screen sizes.

Explanation:

  • Column 1: This column uses classes "col-xl-12 col-sm-6 col-md-3." It will occupy all 12 columns on extra-large screens (xl), 6 columns on small screens (sm), and 3 columns on medium screens (md).
  • Column 2: This column uses only "col-12." It will occupy all 12 columns on extra small screens (xs) by default.
