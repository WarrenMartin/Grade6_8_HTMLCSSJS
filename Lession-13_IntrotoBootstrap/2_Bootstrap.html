What is Bootstrap?

Bootstrap is a popular CSS framework that provides a collection of pre-built styles, components, and JavaScript functionality to streamline web development.

Dependencies

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>

These dependencies are necessary for Bootstrap to work effectively.

Container

A container is a core Bootstrap component used to wrap the page content. It provides a fixed-width container to center and manage content within a webpage. This is an example:

This is content within a container.

Code Snippet

                    <div class="container">
                        <p>This is content within a container.</p>
                    </div>

Container Fluid

A container-fluid, on the other hand, spans the full width of the viewport. It's used when you want your content to occupy the entire available width. This is an example:

This is content within a container-fluid.

Code Snippet

                    <div class="container-fluid">
                        <p>This is content within a container-fluid.</p>
                    </div>
