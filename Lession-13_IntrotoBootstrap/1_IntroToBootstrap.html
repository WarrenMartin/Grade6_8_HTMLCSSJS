Understanding Bootstrap

What is Bootstrap?

Bootstrap is a widely-used CSS framework that simplifies web development by providing pre-designed styles, components, and JavaScript functionality.

Why Use Bootstrap?

  • Consistency: Bootstrap ensures a consistent and professional design for your website.
  • Responsive Design: It makes your site adaptable to various screen sizes, including mobile devices.
  • Rapid Development: Bootstrap offers ready-made components that speed up development.
  • Community and Support: It has a large and active community, making it easy to find help and resources.

Bootstrap Grid System

Bootstrap's grid system is based on a 12-column layout, which is fundamental to creating responsive designs. It uses container, rows, and columns to structure your content.

Containers: These act as the outermost element to contain the layout.

Rows: Rows are used to create horizontal groups of columns. They ensure proper alignment and padding.

Columns: Columns are used to define how content spans across the 12 available columns. They are specified using classes like 'col-sm-4'.
