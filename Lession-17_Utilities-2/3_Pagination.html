Pagination in Bootstrap

Pagination is used to divide content into separate pages and present them in a more organized way. Bootstrap provides several classes to implement pagination easily.

Explanation of Bootstrap Pagination Classes

  • pagination: The main class for pagination that provides basic styling.
  • page-item: Represents an individual page item in the pagination list.
  • page-link: Represents the link inside a page item.
  • disabled: A utility class that disables the page link.

Explanation of ARIA Attributes

  • aria-label: Provides a label or description for an element that is not represented by visible text. In our example, it describes the purpose of the "Previous" and "Next" buttons for screen readers.
  • aria-hidden: Indicates whether an element and its subtree should be visible or hidden to accessibility tools. In our example, it is used to hide the visual representation of the "Previous" and "Next" buttons from screen readers to avoid redundant information.

Code Snippet

HTML Code:

            <nav aria-label="Page navigation example">
                <ul class="pagination">
                    <li class="page-item">
                        <a class="page-link" href="#" aria-label="Previous">
                            <span aria-hidden="true">«</span>
                        </a>
                    </li>
                    <li class="page-item"><a class="page-link" href="#">1</a></li>
                    <li class="page-item"><a class="page-link" href="#">2</a></li>
                    <li class="page-item"><a class="page-link" href="#">3</a></li>
                    <li class="page-item">
                        <a class="page-link" href="#" aria-label="Next">
                            <span aria-hidden="true">»</span>
                        </a>
                    </li>
                </ul>
            </nav>
