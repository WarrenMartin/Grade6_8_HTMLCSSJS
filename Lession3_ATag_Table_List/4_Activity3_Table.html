HTML Tables

Introduction to HTML Tables

HTML tables are used to display data in a structured format. They consist of rows and columns, with each cell containing data. Tables are a fundamental part of web design for presenting information in a grid-like layout.

Creating a Simple Table

Below is a simple example of an HTML table:

Header 1 Header 2
Data 1, Row 1 Data 2, Row 1
Data 1, Row 2 Data 2, Row 2

In this example, the <table> element is used to create the table structure. <tr> defines table rows, <th> is used for table headers, and <td> is used for table data.

Styling Tables

Tables can be styled using CSS to improve their appearance. In this example, we added basic styling to the table for demonstration purposes. You can customize the appearance of your tables as needed.
