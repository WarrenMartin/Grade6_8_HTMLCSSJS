HTML Lists - Ordered & Unordered

Unordered List (ul)

An unordered list is created using the <ul> element. It is used to represent a list of items with no specific order:

  • Item 1
  • Item 2
  • Item 3

Ordered List (ol)

An ordered list is created using the <ol> element. It represents a list of items in a specific order, typically with numbers:

  1. First Item
  2. Second Item
  3. Third Item

Both ordered and unordered lists can contain list items (<li>), and you can nest lists within lists to create hierarchical structures. Lists are commonly used for navigation menus, itemized content, and more.
