Understanding Containers in Bootstrap

A container is a fundamental building block in Bootstrap's grid system.

What is a Container?

In Bootstrap, a container is a wrapper for content on your page. It provides a simple and effective way to center your content and keep it organized.

Types of Containers

Bootstrap has two types of containers:

  • .container: Fixed-width container.
  • .container-fluid: The class container-fluid provides a full width container that spans the entire width of the viewport .
  • .container-breakpoint: which is width 100% until the specified breakpoints.

How to Use Containers

To use a container, simply add the class .container or .container-fluid to a <div> element. 

<div class="container">
    
</div>

<div class="container-fluid">
    
</div>

Responsive Utility Classes

Bootstrap provides responsive utility classes to control the layout at different breakpoints. For example, .col-sm-6 means the column will take 6 units on small screens.
