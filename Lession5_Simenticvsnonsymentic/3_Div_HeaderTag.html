Understanding HTML Tags: div, header, and nav

The <div> Tag

The <div> tag is a non-semantic container that is used to group and style content. It is often used for layout purposes and can contain other HTML elements. Here's an example:

<div id="myDiv">
 <h2>This is a Div Container</h2>
 <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>

The <header> Tag

The <header> tag represents a container for introductory content or a set of navigational links. It often contains the main heading and site logo. Here's an example:

<header>
 <h1>Welcome to My Website</h1>
 <nav>
  <a href="#">Home</a>
  <a href="#">About</a>
  <a href="#">Services</a>
  <a href="#">Contact</a>
 </nav>
</header>

The <nav> Tag

The <nav> tag defines a section of navigation links. It is typically placed within the <header> element to create a navigation menu. Here's an example:

<nav>
 <a href="#">Home</a>
 <a href="#">About</a>
 <a href="#">Services</a>
 <a href="#">Contact</a>
</nav>
