Understanding the Output Tag

The Output Tag

The <output> element is used to represent the result of a calculation or user input. It provides a way to display the output of a script, mathematical calculation, or any other dynamic content on a webpage.

Example: Calculating the Sum

Let's use the <output> element to display the sum of two numbers entered by the user:



Code Snippet

Here's the code snippet for using the <output> tag to display the sum:

            <form oninput="out.value= parseInt(num1.value) + parseInt(num2.value)">
            <label for="num1">Num1:</label>
            <input type="number" name="num1" id="num1">
            <br>
            <label for="num2">Num2:</label>
            <input type="number" name="num2" id="num2">

            <label>Ans</label>
            <output for="num1 num2" id="out"  name="out"></output>

Output

Understanding the "for" Attribute

Example: Using the "for" Attribute


The "for" attribute in the <label> is like a bridge that connects the label ("Number 1") to the input field below it. In other words, it tells the web browser that the label "Number 1" is associated with the input field.
