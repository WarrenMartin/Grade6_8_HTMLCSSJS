Understanding Labels and Buttons

Labels

A label is used to describe a form element, such as an input field. It helps improve the usability and accessibility of a webpage by providing a text description of the associated element. Labels are typically used with form elements like input fields, radio buttons, checkboxes, and more.

Here is an example of a label:

 
        <label >Username:</label>

Buttons

Buttons are interactive elements on a webpage that allow users to perform actions. They can be used to submit forms, trigger JavaScript functions, or navigate to other parts of the website. There are various types of buttons, including:

  • Submit Button: Used to submit a form.
  • Button Element: A general-purpose button for custom JavaScript actions.

Here is an example of a submit button:

 
        <input type="submit" value="Submit">

Code Snippet

Let's put it all together with a simple form that uses labels and a submit button:

 
        <form action="submit.php" method="post">
            <label >Username:</label>
            <input type="text" id="username" name="username">
            <br>
            <label >Password:</label>
            <input type="password" id="password" name="password">
            <br>
            <input type="submit" value="Submit">
        </form>

This code creates a simple form with labels for the "Username" and "Password" fields and a submit button.
