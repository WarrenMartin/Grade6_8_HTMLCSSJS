Gradient

What is Gradient?

A gradient is a smooth transition between two or more colors. It can create beautiful color effects on elements like backgrounds.

Types of Gradients

There are two main types of gradients:

  1. Linear Gradient: It creates a transition in a straight line.
  2. Radial Gradient: It creates a transition in a circular or radial pattern.

Linear Gradient Examples
