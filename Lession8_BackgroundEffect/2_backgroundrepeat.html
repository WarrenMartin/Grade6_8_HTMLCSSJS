Understanding Background Repeat

Background-Repeat Property

The "background-repeat" property defines how a background image should be repeated. Here are four common values:

1. repeat

The background image is repeated both vertically and horizontally. It creates a seamless grid of the image, filling the entire element. This means the image is repeated as many times as needed to cover the entire background.

HTML:

<div class="repeat-box repeat-box-repeat">Repeat</div>

CSS:

.repeat-box-repeat {
    background-image: url('background.jpg');
    background-repeat: repeat;
    width: 200px;
    height: 200px;
}

The background image is repeated both vertically and horizontally. It creates a seamless grid of the image, filling the entire element. This means the image is repeated as many times as needed to cover the entire background.

Repeat

2. round

The "round" value for "background-repeat" repeats the background image, and the last image is resized to fit the remaining space. If the background image doesn't perfectly fit the element, it will be stretched or compressed slightly to make it fit, ensuring there's no empty space.

HTML:

<div class="repeat-box repeat-box-round">Round</div>

CSS:

.repeat-box-round {
    background-image: url('background.jpg');
    background-repeat: round;
    width: 200px;
    height: 200px;
}

The "round" value for "background-repeat" repeats the background image, and the last image is resized to fit the remaining space. If the background image doesn't perfectly fit the element, it will be stretched or compressed slightly to make it fit, ensuring there's no empty space.

Round

3. space

The "space" value for "background-repeat" repeats the background image with space between each repetition. This results in even distribution of the background images, with consistent spacing between them, covering the entire element.

HTML:

<div class="repeat-box repeat-box-space">Space</div>

CSS:

.repeat-box-space {
    background-image: url('background.jpg');
    background-repeat: space;
    width: 200px;
    height: 200px;
}

The "space" value for "background-repeat" repeats the background image with space between each repetition. This results in even distribution of the background images, with consistent spacing between them, covering the entire element.

Space

4. no-repeat

The "no-repeat" value for "background-repeat" ensures the background image is displayed only once and is not repeated. If the element is larger than the image, it won't be stretched or repeated; it just displays once.

HTML:

<div class="repeat-box repeat-box-no-repeat">No Repeat</div>

CSS:

.repeat-box-no-repeat {
    background-image: url('background.jpg');
    background-repeat: no-repeat;
    width: 200px;
    height: 200px;
}

The "no-repeat" value for "background-repeat" ensures the background image is displayed only once and is not repeated. If the element is larger than the image, it won't be stretched or repeated; it just displays once.

No Repeat
