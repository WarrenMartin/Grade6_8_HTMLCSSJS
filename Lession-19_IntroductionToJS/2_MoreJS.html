JavaScript Basics

Variables in JavaScript

A variable in JavaScript is a container for storing data values. Variables are used to store and manipulate information in a program.

Rules for Variables

  • Must start with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can also be digits (0-9).
  • Variables are case-sensitive (e.g., myVariable and myvariable are different).
  • Cannot be a JavaScript keyword.
  • Cannot contain spaces.

Example:

            
                var myVariable = "Hello, World!";
                var _myOtherVariable = 42;

Statements in JavaScript

A statement in JavaScript is a set of instructions that performs a specific task. Statements are terminated by a semicolon (;).

Example:

            
                var x = 5;
                var y = 10;
                var sum = x + y;

Data Types in JavaScript

JavaScript has several data types that define the types of values a variable can hold. The main data types are:

  • String: Textual data, enclosed in single or double quotes.
  • Number: Numeric data, including integers and floating-point numbers.
  • Boolean: Represents true or false.
  • Array: Ordered list of values, enclosed in square brackets.
  • Object: Collection of key-value pairs, enclosed in curly braces.
  • Null: Represents the absence of value.
  • Undefined: Represents an uninitialized variable.

Examples:

            
                var name = "John";
                var age = 30;
                var isStudent = false;

                var numbers = [1, 2, 3, 4, 5];
                
                var person = {
                    firstName: "Jane",
                    lastName: "Doe"
                };

                var emptyValue = null;
                var notDefined;
