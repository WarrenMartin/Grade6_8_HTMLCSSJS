JavaScript Concepts: Regular Expressions and Loops

Regular Expressions

A regular expression, or regex, is a powerful tool for pattern matching in strings. It is defined using a combination of characters that represent a specific pattern.

Example:

We can create a simple regex to match the word "world" in a string:

/world/

Usage in JavaScript:

In JavaScript, the test() method is commonly used with regular expressions to check if a string matches a pattern.

const pattern = /world/;
const stringToTest = "Hello, world!";
const containsWorld = pattern.test(stringToTest);
console.log(containsWorld); // Output: true

Loops

Loops are used in JavaScript to repeatedly execute a block of code. They provide a way to iterate over a set of data or perform a task multiple times.

For Loop:

The for loop is used when the number of iterations is known.

for (let i = 0; i < 5; i++) {
// Code to be repeated
console.log(i);
}

While Loop:

The while loop is used when the number of iterations is not known in advance.

let i = 0;
while (i < 5) {
// Code to be repeated
console.log(i);
i++;
}
