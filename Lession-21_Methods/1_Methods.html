Understanding JavaScript Methods

In JavaScript, methods are functions associated with objects. They can perform actions or provide information related to the object they are called on.

String Methods

toUpperCase()

The toUpperCase() method converts a string to uppercase.

const originalString = "hello";
const uppercasedString = originalString.toUpperCase();
console.log(uppercasedString);

toLowerCase()

The toLowerCase() method converts a string to lowercase.

const originalString = "WORLD";
const lowercasedString = originalString.toLowerCase();
console.log(lowercasedString);

startsWith()

The startsWith() method checks if a string starts with a specified sequence of characters.

const stringToCheck = "Hello, how are you?";
const startsWithHello = stringToCheck.startsWith("Hello");
console.log(startsWithHello);

test()

The test() method is used with regular expressions to test if a string contains a pattern.

const pattern = /world/;
const stringToTest = "Hello, world!";
const containsWorld = pattern.test(stringToTest);
console.log(containsWorld);

Math Methods

Math.random() and Math.floor()

Math.random() generates a random decimal between 0 (inclusive) and 1 (exclusive). Math.floor() rounds a number down to the nearest integer.

const randomDecimal = Math.random(); // Example: 0.764523987
const randomInteger = Math.floor(randomDecimal * 10); // Example: 7
console.log(randomInteger);
