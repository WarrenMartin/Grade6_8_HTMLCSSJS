Pseudo Classes

:hover

Hover over this element

:hover is a pseudo-class used to define styles for an element when a user hovers the cursor over it. It's often used to create interactive and responsive design elements.

:active

Click and hold on this element

:active is a pseudo-class used to define styles for an element when it's being actively clicked or held down, such as clicking a button or link. It provides visual feedback during the click action.

:focus

:focus is a pseudo-class used to define styles for an element when it gains focus, typically through user interaction like clicking or tabbing. It's commonly used to style form elements when they are selected, enhancing accessibility and user experience.
