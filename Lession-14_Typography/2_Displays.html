Display Headings, Small, and DL in Bootstrap

Display Headings

Bootstrap provides display headings to make a bold statement on your page. These are used with classes like .display-1, .display-2, .display-3, and .display-4. 

        
          <div class="display-1">Display 1</div>
          <div class="display-2">Display 2</div>
          <div class="display-3">Display 3</div>
          <div class="display-4">Display 4</div>

Small Element

The <small> element in Bootstrap is used to create smaller, de-emphasized text. It is often used for fine print or disclaimers. 

        
          <p><small>This is small text.</small></p>

This is small text.

Definition List (DL)

The <dl> element in HTML is used to create a description list. In Bootstrap, it can be styled using utility classes. 

        
          <dl class="row">
            <dt class="col-sm-3">Term 1</dt>
            <dd class="col-sm-9">Description 1</dd>
            <dt class="col-sm-3">Term 2</dt>
            <dd class="col-sm-9">Description 2</dd>
          </dl>
