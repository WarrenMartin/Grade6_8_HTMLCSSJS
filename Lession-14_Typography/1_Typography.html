Welcome to the Bootstrap Typography Guide

Typography in Bootstrap

Bootstrap provides a default font size of 16px with a base font size of 1rem. This makes it easy to maintain consistent typography throughout your website.

Headings and Subheadings

Bootstrap offers classes like .h1, .h2, .h3, etc., to define the hierarchy and styling of your text. Customize them to suit your design.

Customizing Headings

Use the Bootstrap heading classes to customize the hierarchy and styling of your text. Here's an example code snippet: 

        
          <h1 class="text-primary">Custom H1 Heading</h1>
          <h2 class="text-success">Custom H2 Heading</h2>
          <h3 class="text-info">Custom H3 Heading</h3>
          <h4 class="text-warning">Custom H4 Heading</h4>
          <h5 class="text-danger">Custom H5 Heading</h5>
          <h6 class="text-secondary">Custom H6 Heading</h6>

Color Types

Use contextual classes like .text-primary, .text-success to convey meaning or highlight content with different colors.

Alignment

Control text alignment with classes like .text-left, .text-center, .text-right.

Code Snippet

Here's an example of how to implement these concepts in HTML: 

        
          <h1 class="text-primary">This is a Primary Heading</h1>
          <h2 class="text-success">This is a Success Subheading</h2>
          <p class="text-muted">This is some muted text.</p>
          <p class="text-center">This text is centered.</p>

Output

This is a Primary Heading

This is a Success Subheading

This is some muted text.

This text is centered.
