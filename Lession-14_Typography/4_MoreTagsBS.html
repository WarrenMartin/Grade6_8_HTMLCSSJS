2. Keyboard Input - <kbd>

3. Preformatted Text - <pre>

The <pre> (preformatted text) tag in HTML is used to define preformatted text. The text inside a 

 element is displayed in a fixed-width font (usually Courier), and it also preserves both spaces and line breaks. This means that the text is displayed exactly as it appears in the HTML code.

        
            
<pre>
    <code>
        <html>
            <head>
                <title>My Web Page</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
            </body>
        </html>
    </code>
</pre>
