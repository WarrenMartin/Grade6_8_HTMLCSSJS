Bootstrap Tags Explanation

The mark tag is used to highlight or mark pieces of text. It can be useful for emphasizing specific content.

The sub tag is used to define subscripted text. For example, H2O represents water with the "2" in a lower position.

The sup tag is used to define superscripted text. For example, E=mc2 represents Einstein's famous equation.

The <code> tag in HTML is like a special box where you can put computer code. It's great for showing a short piece of code or a single line of code in the middle of a paragraph or text.

<html>
<head>
  <title>My Web Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
</body>
</html>
